Installing AWS CLI

1. Download Windows 64-bit installer from:
   http://docs.aws.amazon.com/cli/latest/userguide/awscli-install-windows.html#install-msi-on-windows
   It installs to \Program Files\Amazon\AWSCLI\.
2. The following Environment Variables need to be setup:
     AWS_ACCESS_KEY_ID
     AWS_SECRET_ACCESS_KEY
